#为啥Redis使用skiplist而不是red-black#
1. skiplist和red-black复杂度一样，但是skiplist实现更简单
2. 并发环境下，红黑树插入、删除时可能需要rebalance，可能涉及整个树的其它部分，skiplist的操作更局部性。锁需要关注的节点更少，因此性能更好。
